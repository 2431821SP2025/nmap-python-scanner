# Nmap Python Scanner

This is a simple Python port scanner created for cybersecurity practice.

## Skills Used
- Python
- TCP ports
- Socket programming
- Basic network reconnaissance

## What I Learned
I learned how port scanning works at a basic level and how tools like Nmap identify open services.

Platform:
- VMware Workstation

Programming Language:
- Python 3.12

Purpose:

- Compare results against industry tools such as Nmap
